# Myango Flutter Web (Prebuilt)

This repo contains the prebuilt Flutter web output.

Render (Static Site):
- Publish Directory: `flutter_app/build/web`
- Build Command: leave empty (prebuilt)

If you update the app:
1. Rebuild: `flutter build web` (run inside `flutter_app`)
2. Commit the updated `flutter_app/build/web`
